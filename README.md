# Markdown System

## Overview

This project provides a system for managing and rendering markdown files efficiently.

## Features
- **Fast rendering**: Optimized for speed.
- **Customizable themes**: Easy to integrate UI styles.
- **Extensible**: Add your own markdown extensions.

## Installation

```bash
# Install the system
npm install markdown-system
```

## Usage

```javascript
const Markdown = require('markdown-system');

const doc = new Markdown('path/to/file.md');

doc.render();
```
